# KB Review Queue

This directory stores candidate claim-card review artifacts generated by:

```bash
cgm-agent kb-ingest --pdf path/to/source.pdf --out-dir src/hermes_cgm_agent/knowledge/review_queue --kb-version kb-YYYY-MM
```

Rules:

- Candidate JSON files are not production knowledge.
- Do not move a card into `authoritative_kb.json` with `verified=true` until the source page, numbers, units, population, and translation are checked.
- A verified production card must include `reviewer` or `reviewed_at`; `kb-validate` enforces that provenance gate.
- Keep personal memory out of this directory. The authoritative KB is read-only at runtime.
