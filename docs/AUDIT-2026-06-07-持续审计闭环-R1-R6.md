## 3. 正确性核验（无 bug，本身即结论）

| 子系统 | 核验点 | 结论 |
|--------|--------|------|
| **analytics**（metrics/events） | TIR/TAR/TBR 边界(70/180 含端)、GMI(Bergenstal)、Kovatchev LBGI/HBGI([20,600] 钳制)、空窗 guard、min_episode covered-span 门控、确定性 event_id | ✅ 正确（CV 用总体 SD 为约定，非 bug） |
| **数据管线**（normalizer/importer/domain） | 单位 18.0182 换算、`value_mg_dl` 派生、**要求显式单位无静默兜底**、tz-aware UTC | ✅ 正确（杜绝单位推断型污染） |
| **storage 加密** | Fernet 版本化 token、key/db `0600`、env-key 模式不落文件、legacy 迁移回退 | ✅ 正确（README 声明属实） |
| **dexcom**（tokens/sync/mapper） | token 封存+过期 skew、DB 约束去重、401 单次刷新重试、**mapper 读取 API unit 字段**（AU/OUS mmol 不污染） | ✅ 正确 |
| **reports 安全门** | 红区**整体替换** sections（零泄漏）、不收集记忆候选、黄区前缀；与三区规范一致 | ✅ 正确 |
| **KB 管线**（quality/merge/validator） | 垃圾/PUA/元数据/标题片段拒绝 + id&hash 去重；merge 强制 `verified=false`+`tier=auto`；validator 守 `verified=true 需 provenance`（CI 门禁） | ✅ 正确（"最大技术债"是内容深度=策展工作，非代码） |
| **持久层**（data/repository） | PHI 列 seal/unseal 对称；WHERE/ORDER BY 仅用明文列（user_id/timestamp/...），无加密列范围查询 | ✅ 正确 |
| **memory**（assembler/review） | 双轨证据 kind 标注（user_memory/authoritative_kb，从不合并）；候选 auto-accept 仅当无需确认、crash-safe accept→mark 顺序 | ✅ 正确 |
